Simple Todo App

A simple **Full Stack Todo application** built with **Flutter** and **GetX** state management. This project demonstrates CRUD operations, state management, navigation, and responsive UI design.

Features
- Add, edit, and delete todo tasks.
- Mark tasks as completed.
- Real-time UI updates using **GetX state management**.
- Clean and responsive interface.

Technologies Used
- Flutter
- GetX (State Management)
- Dart
